About this project

I built this website over the course of two weeks. I wanted to build an application using Flask, and I stumbled upon a Flask video series by Corey Shafer.
Completing the series and building this blog became an outlet for experimenting with HTML/CSS, Python, SQL and Bootstrap.

What I learned from this project

  • Bootstrap navigation, alerts, and styling.
  • Using SQL to store, retrieve, and update user information, posts, and passwords.
  • Breaking a project into templates, packages, and forms so that debugging isn't a total nightmare.
  • Setting up a linux server, and deploying an application.

Features

  1. Create posts
  2. Register,login
  3. Reset passwords
  4. Edit posts(if logged in)
  5. Delete posts
  6. Set a profile picture

What I will do differently moving forward

  1. Organize the files into packages at the beginning so that I don't have to refactor so many forms/templates
  2. Build out a layout page and use extension blocks so I do not repeat as many code blocks
  3. Take more breaks when I feel stuck or discouraged during debugging
  4. Drink more coffee :)
